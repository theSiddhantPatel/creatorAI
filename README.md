# CreatorAI
CreatorAI is a full-stack web application designed to empower creators with AI-driven tools for content generation, automation, and collaboration.  
Built with **Next.js**, **TailwindCSS**, **Shadcn UI**, and **ImageKit**, the platform provides a modern, scalable, and accessible experience.
